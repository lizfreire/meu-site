# 🌍 Diário de Viagem

Este projeto foi desenvolvido como atividade prática utilizando HTML, CSS e JavaScript.

## ✨ Funcionalidades
- Navegação entre páginas
- Galeria de fotos organizada
- Layout simples e intuitivo

## 📸 Sobre o projeto
O site apresenta um diário de viagem com registros de momentos especiais, incluindo locais visitados e experiências vividas.
